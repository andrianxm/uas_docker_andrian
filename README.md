Proyek UAS Cloud Computing: Implementasi Microservices dengan Docker & Redis.

## Fitur
- **AES-128 Encryption**: Mengamankan data sensitif menggunakan algoritma standar industri.

## Teknologi
- **Backend**: Python Flask
- **Database**: Redis (Alpine Image)
- **Container**: Docker & Docker Compose
- **Security**: PyCryptodome (AES-128 ECB)
